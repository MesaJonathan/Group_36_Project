What is Schedule Help Pro?

Schedule Help Pro is a website that makes the registration process for students easier and more efficent.
This website provides students with access to various resources such as rate my professor reviews, avaiable
classes, and available tutors. By using this website, students will not have to open multiple tabs in order
to register. Rather they will be able to have all the materials they need for registration in one place.

How to use Schedule Help Pro

You can access the resources Schedule Help Pro provides by registering through the login page
on our website. If you already have an account you can click on the login page and enter your
information. After logging in, you will be redirected to a class list page which will show
you a class list. You can then click on a class to access the page for it
